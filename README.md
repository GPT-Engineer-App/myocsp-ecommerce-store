# myocsp-ecommerce-store

An ecommerce store for an OCSP on-demand service called MyOCSP. The application uses OpenAPI for API documentation, MySQL as the primary database, and Redis for caching.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/myocsp-ecommerce-store.git
cd myocsp-ecommerce-store
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
